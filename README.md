
# Digital Marketing Platform
This project is a Work In Progress. This project is made as part of the final year engineering project.




## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```
Go to Client and Server directory
```bash
cd digital_client
cd digital_server
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
